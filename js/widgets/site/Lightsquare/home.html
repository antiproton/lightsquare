Fast and reliable live chess

Lightsquare is the client reference implementation of the jsonchess protocol. It is designed for simplicity and robustness, and makes use of the full capabilities of modern browsers to provide a nice interface and minimise latency.

All of the code for jsonchess, this web application, the server it connects to, and the chess rules library is free and open source, and can be found on Github.

To start a game, click Play above and then:

  • accept a challenge by clicking it in the graph, or
  • create a seek by filling in the form.

Players named ‘Stockfish N’ are bots placed on the server for testing purposes.
