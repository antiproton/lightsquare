Live chess

Lightsquare is a live chess server for person-to-person play. Registration is recommended for keeping track of your rating, but not required. Unlimited games can be played simultaneously.

Open platform

Lightsquare is a collection of open-source projects. Anyone can download, modify, extend and re-distribute the code.

Get the code:

  • Chess rules and game logic
  • Backend
  • Frontend (this website)
